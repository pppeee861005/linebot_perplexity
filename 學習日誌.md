# Git 學習日誌

## 2025-12-24

### 今天學習了 Git 的真諦

非常快樂！發現 Git 的核心概念：

#### 🌌 分支 (Branch) = 平行宇宙
- 每個分支都是獨立的開發空間
- 可以平行發展而不互相干擾
- 就像科幻小說中的平行宇宙理論

#### 🏷️ 版本標籤 (Tag) = 宇宙里程碑
- 分支之下還有 v1.0, v2.0 等版本
- 標記重要的穩定版本
- 永久記錄專案的重要時刻

#### 💡 個人體悟
- Git 不只是版本控制工具，更是時間旅行和宇宙探索的工具
- 每個 commit 都是時間軸上的節點
- 每個分支都是可能的未來
